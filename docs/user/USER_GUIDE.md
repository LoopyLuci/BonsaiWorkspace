# Bonsai Workspace User Guide
## Quick Start
1. Install: download from releases or build with `BonsaiExeLauncherBuilder.ps1`
2. Launch: `BonsaiWorkspace.exe` or `node launch-all.mjs --mode desktop`
3. Load a model: Model Selector → pick any GGUF → wait for "Ready"
4. Chat: type a message and press Send
## Code Generation
Type "Create a Python script that prints Hello World" — the AI generates code, writes files, and you can Run them in the sandbox.
## Running Code
Click the Run button on any code block. Choose Python (venv) for full Python, or JS (WASM) for safe sandboxed execution.
## BonsAI Lab
Compare your fine-tuned adapter against a reference model. Run continuous training loops to improve BonsAI automatically.
## Settings
- Models: load/unload, check VRAM usage
- Features: toggle experimental features on/off
- Training: fine-tune adapters on your own data
## Getting Help
See CONTRIBUTING.md for development setup. Report issues on GitHub.
